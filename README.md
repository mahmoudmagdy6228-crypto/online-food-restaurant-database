# online-food-restaurant-database
# Online Food Restaurant Management System 🍔

A relational database project designed for managing an online food restaurant system using SQL and database design concepts.

## 📌 Project Overview

This project focuses on building a structured database system for handling:

* Customers
* Orders
* Products
* Restaurants
* Order Details

The system demonstrates how real-world restaurant platforms manage data relationships and transactions efficiently.

* Entity Relationship Diagram (ERD)
* Relational Schema Mapping
* SQL Database Implementation
* Primary & Foreign Keys
* Relationship Constraints
* Real-world Database Modeling
